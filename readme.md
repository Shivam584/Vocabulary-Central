## Introduction
It is website to automate your schedules as reminder notification through mails at specified time date.It is implemented using <b>Django</b> for Backend Request-Response cycle, <b>HTML & CSS</b> for Frontend design, <b>Redis</b> for caching purpose and <b>MySQlite3</b> for Database.
